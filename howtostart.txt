Требуется:

Visual Studio

Visual Code с поддержкой TypeScript



Запуск основного проекта:

1. Открыть csharpgame.sln

2. Выполнить "Восстановить пакеты через nuget"

3. Свойства решения > Запускаемый проект > Несколько запускаемых проектов > Выбрать wcfservice и webapi

4. Запустить решение F5



Запуск Angular 8 UI:

1. Открыть папку .\angular в Visual Code

2. Выполнить команду npm i

3. Выполнить команду ng serve

4. Открыть http://localhost:4200/
